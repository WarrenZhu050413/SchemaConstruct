innerText Output Examples

1. Basic Block Elements

<div> <h2>Gradient Descent</h2> <p>Learns by following the negative gradient.</p> </div>
Gradient Descent\nLearns by following the negative gradient.

Line breaks inserted between block-level elements.

2. Inline Elements & Hidden Span

<p>Adam combines <strong>momentum</strong> and adaptive rates.<span style="display:none">Hidden text</span></p>
Adam combines momentum and adaptive rates.

Hidden span ignored; inline elements merged into one line.

3. List with Nested Elements

<ul> <li>Initialize θ</li> <li>Repeat until convergence: <ul> <li>Compute gradient</li> <li>Update parameters</li> </ul> </li> </ul>
• Initialize θ\n• Repeat until convergence:\n • Compute gradient\n • Update parameters

Bullets rendered using the browser’s list markers (characters may vary by user agent).

4. Table Content

<table> <tr><th>Epoch</th><th>Loss</th></tr> <tr><td>1</td><td>0.45</td></tr> <tr><td>2</td><td>0.32</td></tr> </table>
Epoch\tLoss\n1\t0.45\n2\t0.32

Tables emit tab-separated columns with newline-separated rows.

5. Script & Style Skipped

<div> <p>Learning Rate: 0.001</p> <style>.hidden { color: red; }</style> <script>console.log('debug');</script> </div>
Learning Rate: 0.001

`innerText` omits the contents of <style> and <script> tags.
