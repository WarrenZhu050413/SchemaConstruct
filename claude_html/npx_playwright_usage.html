üìç Setup Essentials ‚ñ∂

Before Running

  • Install Browsersnpx playwright install chromium firefox webkit
  • Verifynpx playwright --version
  • Dependenciesnpx playwright install-deps (Linux)
Why install browsers?

Playwright ships browser binaries per release. Run install after upgrades or fresh clones to ensure versions match the current CLI.

CI tip

Add npx playwright install --with-deps to CI provisioning scripts so test runners have browsers ready.
